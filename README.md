## Project ZERO-1

Autonomous AI system that automatically finds, exploits, and fixes network security vulnerabilities. Uses LLMs to analyze Cisco configurations, proves exploits work, generates safe patches, and creates cryptographically-secured proof of remediation. Full closed-loop automation for DISA STIG compliance.
